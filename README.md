Tailwind Site

A simple and visually appealing website built using Tailwind CSS and enhanced with AOS (Animate On Scroll) animations.
This project was created to get familiar with Tailwind's utility-first CSS approach and to experiment with smooth scroll-based animations.

-> Features

Tailwind CSS for modern, responsive, and clean UI design.

AOS (Animate On Scroll) integration for smooth animations as elements come into view.

Fully responsive layout across different devices.

Lightweight and beginner-friendly structure.
